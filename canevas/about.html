How to create a new canvas

For example a new level named “L3ML”

  1. make a copy of canevas template:
  1. html file: copy canevas/templates/indexL3ML.html
  2. html file: copy canevas/templates/majq_L3ML.js make copy and put it on canevas directory and canevas/js for js files
  1. Test if it works
  2. Update configuration modules and unites in majq_L3ML.js file
  3. You can add an argument to annee.create_canevas(true) as true to check your canevas
  4. If it works: Add new menu item “L3ML” into file :canevas/js/utils.js function create_menu:
